# memo-rails

## 概要
Rails × Vue.js で簡単なメモアプリを作成
Webpackerを使って、RailsとVue.jsを組み合わせたWebアプリケーションになります。

## このアプリの作成を通して学んだこと

- Webpackerの導入と基本
- APIについて
- Axiosを使った非同期通信
- Vue.jsの基礎
